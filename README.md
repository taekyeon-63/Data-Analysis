# 프로젝트 개요

글로벌 방위산업 수출 전략 수립을 위한 국가별 맞춤형 정보 제공 플랫폼


# 목적

각 국가별로 방위산업과 관련되는 지표를 취합하여 해당 국가에 수출을 목적으로 하는 고객에게 정보를 제공

# 프로젝트 배경

증가하는 국내 방위사업 수출에 따른 국가별 맞춤형 정보를 한눈에 보이게 제공
